## 摘要（据 arXiv 整理）

LLM 擅长泛化，但**无法可靠求解长程（long-horizon）规划问题**。LLM+P 将自然语言描述的问题转换为 **PDDL**（经典规划领域语言），交由**经典规划器**求最优解，再把解翻译回自然语言。实验显示该方法在多数基准上能取得最优解，而纯 LLM 对大多数问题甚至给不出可行计划。

## 核心思想

"LLM 做翻译，经典规划器做规划"：用 LLM 把自然语言问题 ↔ PDDL 互译，把求解外包给可保证最优性的符号规划器——典型的**神经-符号（neuro-symbolic）混合**思路。

## 在 Lilian Weng 框架中的位置

属 **Planning → Task Decomposition**：代表"借助外部经典规划器"的规划增强路线，与 CoT/ToT 的"纯 LLM 内部推理"形成对照。

## 意义

直面 LLM 长程规划不可靠的痛点，示范了用外部符号工具弥补 LLM 规划短板的范式，是 Agent 规划走向神经-符号混合的代表工作。
