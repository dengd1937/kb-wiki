## 摘要（据 arXiv 整理）

AutoGen 是一个开源框架，使开发者能通过**多个可交互的智能体**构建 LLM 应用。这些智能体**可定制、可对话（conversable）**，运行方式是大语言模型、人类输入与工具的组合。开发者可用**自然语言或代码灵活定义智能体的交互行为与对话模式**。该框架可作为数学、编码、问答、运筹、在线决策、娱乐等多领域应用的基础设施。

## 核心思想

把"单 Agent 调工具"升级为**"多 Agent 通过对话协作"**：

- **Conversable Agents**：每个 agent 可发/收消息、维护状态；可由 LLM、人类、工具任意组合驱动。
- **可编程对话**：交互模式（谁和谁说、何时终止、是否引入人类）用自然语言或代码声明式定义。
- 内置 UserProxyAgent / AssistantAgent 等，支持代码执行、群聊（GroupChat）、人类在环。

## 在工程化线中的位置

属 Function Calling 之后的**框架浪潮**阶段：把可靠的单次工具调用编排为**多智能体对话系统**，是"多 Agent 协作"工程范式的代表，也是后续角色分工编排（Planner/Executor/Critic）的先声。
